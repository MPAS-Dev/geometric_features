geometric_features.FeatureCollection¶

class geometric_features.FeatureCollection(features=None, otherProperties=None)[source]¶

An object for representing and manipulating a collection of geoscientific geometric features.

Variables

  • features (list of dict) – A list of python dictionaries describing each feature, following the geojson convention

  • otherProperties (dict) – Other properties of the feature collection such as type and groupName

__init__(features=None, otherProperties=None)[source]¶

Parameters

  • features (list of dict, optional) – A list of python dictionaries describing each feature, following the geojson convention
